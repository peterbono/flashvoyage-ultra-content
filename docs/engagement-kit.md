# FlashVoyage — Kit Engagement IG (30 min/jour)

> Objectif : briser le cold start IG (1 follower) en créant de la visibilité organique
> via des interactions authentiques sur les comptes voyage FR établis.
> Installé le 2026-04-07.

## Pourquoi c'est la priorité #1

L'algorithme IG en 2025-2026 récompense les comptes qui **interagissent** avec la communauté,
pas ceux qui publient dans le vide. 16 reels publiés / 1 follower = le compte est invisible
car il n'existe pas dans le graphe social IG. Commenter sur les gros comptes FR = apparaître
dans leur fil de commentaires → leurs followers voient @flashvoyage → clicks sur le profil → follows.

## Templates de commentaires (ton "Histoires Vraies")

> Règle d'or : **jamais de lien, jamais de promo, jamais de "viens voir mon profil"**.
> L'algo IG flagge les commentaires promotionnels et les shadowban.
> Le but : être UTILE, AUTHENTIQUE, et VISIBLE.

### 1. La question authentique
```
"T'y es allé en quelle saison ? Je planifie un trip en [mois] et j'hésite encore"
```

### 2. L'anecdote perso courte
```
"J'ai eu exactement le même choc à [lieu] ! Le pire c'est quand tu réalises que [détail spécifique]..."
```

### 3. Le complément d'info utile
```
"Fun fact pour ceux qui lisent : le visa [pays] a changé en 2024, c'est passé à [durée]. Ça change la donne pour les longs séjours"
```

### 4. La réaction émotionnelle (courte)
```
"Cette photo de [lieu] 😍 j'ai passé 3 jours là-bas, la lumière au lever c'est un autre monde"
```

### 5. Le micro-désaccord constructif (engagement bait)
```
"Perso je préfère [alternative] à [lieu du post] pour ça — moins de touristes, moitié prix, et la bouffe de rue est folle"
```

### 6. Le prix choc
```
"Le massage à [prix] alors qu'en France c'est [prix FR]... c'est pour ça que je peux plus rentrer 😅"
```

### 7. Le conseil non sollicité mais utile
```
"Si tu repasses, teste [lieu/resto/activité spécifique]. C'est pas dans les guides et c'est le meilleur [chose] que j'ai trouvé là-bas"
```

### 8. L'empathie voyageur
```
"La galère des [problème voyage courant] 😂 la première fois que ça m'est arrivé à [lieu], j'ai [réaction]. Maintenant je [solution]"
```

### 9. Le saviez-vous
```
"Ce que beaucoup savent pas c'est que [info pratique surprenante sur le lieu/pays]. Ça m'a sauvé [montant/temps] la dernière fois"
```

### 10. La validation + rebond
```
"100% d'accord. Et le truc que personne mentionne c'est [complément]. C'est ce qui fait VRAIMENT la différence sur place"
```

## Routine quotidienne

### Quand ?
- **8h00-8h30 Bangkok** (= 3h00-3h30 Paris, les posts FR du soir sont encore frais)
- OU **20h00-20h30 Bangkok** (= 15h-15h30 Paris, posts du midi encore actifs)

### Combien ?
- **10 commentaires/jour minimum**
- Répartition : 3 sur Tier 1, 4 sur Tier 2, 3 sur Tier 3
- Cibler les posts de **moins de 12h** (les commentaires early ont plus de visibilité)

### Comment ?
1. Ouvrir l'app IG
2. Aller sur le 1er compte de la liste du jour (rotation sur les 20)
3. Trouver le post le plus récent (< 12h si possible)
4. Lire le post/caption en 10 secondes
5. Adapter un template ci-dessus (PERSONNALISER, ne pas copier-coller mot pour mot)
6. Poster le commentaire
7. **Liker 2-3 commentaires d'autres personnes** sur le même post (signaux sociaux)
8. Passer au compte suivant
9. **Temps total : 2-3 min par compte × 10 = ~25-30 min**

### Tracking
Google Sheet simple avec colonnes :
- Date | Compte | Post commenté | Commentaire résumé | Réponse reçue (oui/non) | Nouveaux followers dans les 24h

### Ce qu'il NE FAUT PAS faire
- ❌ Mentionner @flashvoyage dans les commentaires
- ❌ Mettre un lien vers le site/profil
- ❌ Copier-coller le même commentaire sur plusieurs posts
- ❌ Commenter "Super photo ! 🔥" (signal spam → shadowban)
- ❌ Commenter des posts de plus de 48h (faible visibilité)
- ❌ Faire plus de 30 commentaires/jour (IG rate-limite et flag)

**Signal de succès** : si après 2 semaines tu n'as pas dépassé 30 followers,
le problème n'est pas l'engagement — c'est le profil IG lui-même (bio, highlights,
grille) qui ne convertit pas les visiteurs en followers. À ce stade, auditer le profil.
